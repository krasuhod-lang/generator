# Прототип AI-помощника «Выручай-Сенсей»

Интерактивный прототип чат-виджета для сервиса подбора микрозаймов **«Выручай»**, реализующий техническое задание из issue. Прототип — статический сайт (HTML / CSS / vanilla JS), его можно открыть локально без сборки и без бэкенда.

> **Важно:** это front-end-прототип, демонстрирующий **UX/UI** и **сценарии диалога** из ТЗ. NLP/LLM/RAG/брокеридж представлены **mock-реализациями** на клиенте, чтобы можно было «пощупать» поведение Сенсея до подключения реальных сервисов (YandexGPT / GigaChat / OpenAI + pgvector + API-шлюз МФО).

---

## 🚀 Как запустить

Никакой сборки не требуется. Достаточно одного из вариантов:

```bash
# Вариант 1: прямо открыть файл в браузере
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows

# Вариант 2: поднять локальный статик-сервер
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

## 🗂 Структура (v2 — Phase 0 рефакторинг)

```
.
├── index.html              # Лендинг + разметка виджета и чата
├── styles.css              # Все стили (включая v2-карточки и тикер)
├── app.js                  # Bootstrap: главное меню v2, маршрутизация, magic-link
├── core/
│   ├── state.js            # Глобальное состояние + session_id (UUID)
│   ├── util.js             # Форматирование, аннуитет, escape
│   ├── kb.js               # Mock-RAG: знания по всем категориям
│   ├── nlp.js              # Rule-based intent + parseAmount/parseTerm
│   ├── render.js           # Баблы, typing, ачивки, карусель, реестр карточек
│   ├── dialog.js           # Schema-driven движок шагов
│   └── api.js              # getOffers / getMagicLink (mock ⇄ /api/v1/*)
├── categories/
│   ├── loans.js            # 💰 Микрозаймы (Sprint 1, baseline)
│   ├── credits.js          # 🏦 Потребкредиты (§1.2) + калькулятор переплаты
│   ├── mortgages.js        # 🏠 Ипотека (§1.3) + auto-програмы + сравн. таблица
│   ├── cards.js            # 💳 Кредитные карты (§1.4) + годовой кэшбэк
│   ├── insurance.js        # 🛡 ОСАГО/КАСКО/ДМС/... (§1.5) с под-роутером
│   ├── deposits.js         # 📈 Вклады (§1.6) + анимированный «Счётчик дохода»
│   └── refinancing.js      # 🔄 Рефинансирование (§1.7) + точка безубыточности
└── README.md
```

Каждая категория самодостаточна и состоит из:
* `qualification_schema` — декларативный список шагов;
* `buildOffers(params, profile)` — mock-генератор офферов;
* `cards.register(category, fn)` — рендерер карточки.

Чтобы добавить новую категорию — создайте файл в `categories/` и подключите в `index.html`.

## 🤖 NLP-парсер (mock)

Распознаёт русскоязычные формулировки без LLM, чтобы прототип «понимал» свободный ввод:

| Ввод пользователя | Что распознаётся |
|---|---|
| `Мне нужно 30 тысяч до зарплаты` | сумма = 30 000 ₽, срок = 30 дн., интент = `loan_request` |
| `хочу 10к на неделю` | сумма = 10 000 ₽, срок = 7 дн. |
| `дай десятку` | сумма = 10 000 ₽ |
| `как погасить досрочно?` | интент = `faq` → ответ из базы знаний |
| `позови оператора` | интент = `faq` → перевод |

Парсеры: `parseAmount`, `parseTerm`, `detectIntent` в `app.js`.

В продакшене эта логика заменяется вызовом LLM (YandexGPT / GigaChat / OpenAI) с **системным промптом из §3.2 ТЗ** и function-calling в `get_loan_offers()`.

## 🧠 Системный промпт (для интеграции с LLM)

Готовый текст промпта, соответствующий §3.2 ТЗ, должен быть передан в LLM при каждом запросе:

```text
Ты — Сенсей, умный и дружелюбный финансовый помощник сервиса "Выручай".
Задача: помогать пользователям подбирать микрозаймы и консультировать
по финансовым вопросам.

Тон: вежливый, поддерживающий, элементы восточной философии (Дзен,
путь самурая, мудрость), без перебора. Обращение на "Вы".

Правила:
1. Никогда не обещай 100% одобрение. Говори "высокий шанс одобрения".
2. Условия продукта — ТОЛЬКО из базы знаний (RAG). Если ответа нет —
   предлагай перевести на оператора. Галлюцинации недопустимы.
3. После сбора {Сумма, Срок, Наличие текущих просрочек} вызывай
   функцию get_loan_offers(amount, term_days, overdue).
4. Никогда не запрашивай паспортные данные в чате. Для оформления —
   выдавай magic-link в защищённый ЛК.
```

## 🧪 Сценарии для проверки прототипа в браузере

1. **Свободный ввод** — кликнуть «Поговорить с Сенсеем», написать `Мне нужно 30 тысяч до зарплаты`. Бот распознает сумму и срок и сразу спросит про просрочки.
2. **Quick replies** — пройти flow только кнопками: `Нужны деньги` → `30 000 ₽` → `30 дней` → `Нет просрочек` → `26–40` → `Найм` → карусель офферов.
3. **FAQ** — спросить `как погасить досрочно?` или `какая ставка?` — ответ из базы знаний с кнопкой follow-up.
4. **Реабилитация КИ** — нажать `📈 Прокачать КИ` или ответить `Есть, более 90 дней` — Сенсей предложит «Программу реабилитации» вместо отказа.
5. **Magic-link** — нажать «Оформить в 1 клик» в любой карточке оффера — Сенсей сгенерирует ссылку на защищённый ЛК.
6. **Геймификация** — пройти flow до конца: появятся ачивки `🎯 Первая консультация`, `⚔️ Чистая КИ`, бейдж уровня в шапке чата сменится на «Самурай».
